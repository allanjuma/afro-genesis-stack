üîç Troubleshooting

FUSE Installation Issues

# Check FUSE installation
ldconfig -p | grep libfuse

# Manual installation
sudo apt-get install fuse libfuse2 # Ubuntu/Debian
sudo dnf install fuse fuse-libs # Fedora
sudo pacman -S fuse2 # Arch
sudo zypper install fuse libfuse2 # openSUSE

Node.js Version Issues

# Check Node.js version
node --version

# Install Node.js v18 via nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install 18
nvm use 18

Build Failures

# Clean build
rm -rf node_modules dist AfroNetwork.AppImage
./setup.sh --appimage-only

# Check build logs
npm run build # Manual build for debugging

üîß Advanced Configuration

Custom Port

Modify the launcher script to use a different port:

# Extract AppImage
./AfroNetwork.AppImage --appimage-extract

# Edit usr/bin/afro-network
# Change: python3 -m http.server 8080
# To: python3 -m http.server 3000

# Repackage
appimagetool squashfs-root AfroNetwork-Custom.AppImage
