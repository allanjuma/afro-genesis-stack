Address Generation System

1

Request Generation

User requests new address generation with their MSISDN phone number

Target: afro:[MSISDN]:[???]
2

Brute-Force Search

Validator performs computational search to find valid address:

  • Generates random 32 hex characters
  • Validates Ethereum address compatibility
  • Ensures secp256k1 curve compliance
  • Up to 1,000,000 attempts per generation
3

SMS Validation

Once valid address found, SMS validation is triggered:

  • Extract OTP from extra characters (SHA256 + 6 digits)
  • Send SMS to user's phone number
  • Include both OTP and extra characters
  • User confirms ownership
4

Address Delivery

Validated address is delivered to user:

  • Full address: afro:[MSISDN]:[extra_chars]
  • Protocol compatible: 0x[extra_chars]
  • Ready for blockchain operations
  • Private to user only

Implementation Details

Address Generation Process

#!/bin/bash
# Generate address for MSISDN
generate_afro_address() {
    local msisdn=$1
    local prefix="afro:${msisdn}:"
    local max_attempts=1000000
    
    while [ $attempt -lt $max_attempts ]; do
        extra_chars=$(openssl rand -hex 16)
        candidate_address="${prefix}${extra_chars}"
        eth_address="0x${extra_chars}"
        
        if geth account validate-address "$eth_address"; then
            send_sms_validation "$msisdn" "$extra_chars"
            echo "$candidate_address"
            return 0
        fi
        
        attempt=$((attempt + 1))
    done
}

SMS Validation Function

#!/bin/bash
send_sms_validation() {
    local phone_number=$1
    local validation_code=$2
    local otp=$(echo -n "$validation_code" | sha256sum | cut -c1-6)
    
    curl -X POST "$AFRO_SMS_API_URL" \
        -H "Content-Type: application/json" \
        -d "{
            \"to\": \"${phone_number}\",
            \"message\": \"Your Afro address validation code: ${otp}\",
            \"type\": \"address_validation\"
        }"
}
